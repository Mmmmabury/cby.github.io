GitHub 不记录 Contributions

发表于 2016-03-11   |  

今天向 GitHub push 了下代码，发现它没有记录这几天的 push，百度了一下，发现应该是本地 git 邮件设置的问题，在.gitconfig 文件中，将 email 改为 GitHub 中设置的邮箱便可以。

官网说明

Issues and pull requests

Issues and pull requests 满足一下条件将显示在你的图表中：

  1. 他们在过去一年中操作的
  2. 是独立的库而不是 fork 的库

提交需要满足的条件

提交修改需要满足：

  1. 在过去一中操作的
  2. 电子邮件是和 gitHub 账户 绑定的（在设置中可以添加）
  3. 是在独立的库中提交，而不是 fork
  4. 需要在默认的分支中提交，一般是 master。

另外，必须至少满足以下条件之一

  1. 你是合作者，或者是组织一员
  2. 你已经 fork 这个库
  3. 你已经有 pull 或 issue
  4. 你 star 了这个库
