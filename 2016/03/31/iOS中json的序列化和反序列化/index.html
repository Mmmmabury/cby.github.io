iOS 中 JSON 的序列化和反序列化

发表于 2016-03-31   |   分类于 iOS   |  

在 iOS 编程中会遇到和服务器的通信，服务器会返回一个 JSON 文件，在 iOS 中是用 NSData 来容纳接受的数据的，这时需要将 NSData 反序列化到相应的数据结构。官方以及提供了简单的序列化类，可以直接用官方库中的 NSJSONSerialization 类。

反序列化方法

将接收到了二进制数据转对应的数据结构，可以用：

+ JSONObjectWithData:options:error:
+ JSONObjectWithStream:options:error:

这两个方法会自动解析接收到的数据，并返回对应的结构。

序列化方法

将数据结构装换为二进制串，可以用：

+ dataWithJSONObject:options:error:
+ writeJSONObject:toStream:options:error:
