安装 pip 遇到 "zlib not available pip"

发表于 2016-03-13   |   分类于 Python ， 实践记录   |  

今天在阿里云（centos系统）上安装 pip3 时遇到了 “zlib not available pip”错误，网上查了很久，发现只能重新编译安装过 Python3。步骤如下：

  • 安装 zlib
  • 配置 python 编译设置

修改编辑Modules/Setup.dist文件，将
#zlib zlibmodule.c -I$(prefix)/include -L$(exec_prefix)/lib -lz这行的注释去掉，再进行编译安装。

之后再进行python3.5 get-pip.py就不会出现”zlib not available pip”错误了。
